### Basic question words

- *ne* / *nelik*     ----- what

- *kim*     ----- who

- *qaj* / *qajsï* / *qaj qajsï*   ----- which (in text **qay**, **qaysı**))

- *qalay* (in text) / *qalaj* (Wurm1951:573 Bsp.)  / *hasıl* (40011008)   ----- how 

- *qanša* / *qanše*     ----- how many (in text **qansha**)

- *qa.jda*     ----- where
